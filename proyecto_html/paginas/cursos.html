Nuestra oferta académica

1. Fundamentos y Lógica

  • Linux y la Terminal: Aprenderás a manejar el sistema operativo mediante comandos.
    Sabrás cómo gestionar archivos, permisos, procesos y automatizar tareas sin usar una interfaz gráfica, algo esencial para administrar servidores.
    Disponible próximamente.

  • Python: Dominarás la sintaxis de uno de los lenguajes más versátiles.
    Aprenderás sobre variables, bucles, funciones y cómo usar bibliotecas para análisis de datos o automatización.
    Detalles del curso

  • Principios S.O.L.I.D.: Aprenderás un conjunto de 5 reglas de diseño para escribir código "limpio".
    El objetivo es que tu software sea fácil de mantener, escalar y reutilizar a largo plazo.
    Disponible próximamente.

  • Git y GitHub: Aprenderás el control de versiones.
    Esto te permite guardar "historiales" de tu código, trabajar en equipo sin borrar el trabajo de otros y subir tus proyectos a la nube para compartirlos.
    Disponible próximamente.

2. Desarrollo Web (Frontend)

  • HTML5: Es el "esqueleto" de cualquier sitio web.
    Aprenderás la estructura de las páginas web (títulos, párrafos, imágenes, enlaces).
    Detalles del curso

  • CSS: Dotarás de estilo a las páginas web.
    Aplicarás colores, fuentes, espaciados y cómo hacer que una web se vea bien tanto en computadoras como en celulares (diseño responsivo).
    Detalles del curso

  • JavaScript: El motor que da vida al navegador.
    Aprenderás a darle interactividad a la web (botones que hacen cosas, formularios que se validan, animaciones).
    Detalles del curso

  • React: Libreria basada en "componentes" reutilizables.
    Crearás interfaces de usuario modernas de forma rápida y organizada.
    Disponible próximamente.

  • TypeScript: Para proyectos grandes y profesionales.
    Aprenderás una versión avanzada de JavaScript que ayuda a detectar errores antes de ejecutar el código.
    Disponible próximamente.

3. Especialidades y Backend

  • Streamlit: Aprenderás a crear aplicaciones web para ciencia de datos o inteligencia artificial usando solo Python, sin necesidad de saber HTML o CSS profundo.
    Disponible próximamente.

  • SQL: Lenguaje para hablar con las bases de datos.
    Arprenderás operaciones CRUD: cómo guardar, consultar, actualizar y borrar información de forma estructurada.
    Detalles del curso

  • Django: Framework de Python.
    Aprenderás a crear el "cerebro" de sitios web complejos (backend) de forma segura y rápida, incluyendo sistemas de usuarios y paneles de administración.
    Disponible próximamente.

  • Wordpress: Aprenderás a crear sitios web, blogs o tiendas online usando un gestor de contenidos, sin necesidad de escribir todo el código desde cero.
    Disponible próximamente.

4. Finalización del proyecto

  • Despliegue (Deploy): Aprenderás a sacar tu proyecto de tu computadora y ponerlo en Internet (en la nube) para que cualquier persona en el mundo pueda visitarlo.
    Detalles del curso

  • SEO: Aprenderás estrategias para que tu página web aparezca en los primeros resultados de búsqueda de Google, optimizando el contenido y la velocidad del sitio.
    Disponible próximamente.
