JavaScript: Dinamismo y Lógica en el Navegador

Crea experiencias interactivas y funcionales en tiempo real

JavaScript es el lenguaje esencial que da vida a la web moderna, permitiendo transformar sitios estáticos en aplicaciones dinámicas capaces de reaccionar de forma inteligente a las acciones del usuario.

En esta formación de Conquer Academy, estudiarás desde los fundamentos de la lógica de programación y la sintaxis moderna (ES6+), hasta la manipulación experta del DOM para modificar el contenido y el estilo de la página sin necesidad de recargar el navegador.

Aprenderás a gestionar eventos, realizar peticiones asíncronas a APIs externas mediante Fetch y estructurar tu código de forma modular y profesional, adquiriendo las competencias técnicas necesarias para desarrollar las funcionalidades interactivas que son el estándar actual en la industria tecnológica global.

Temario del curso

  1. Bases del Lenguaje y Lógica
    • Variables (let, const), tipos de datos y operadores.
    • Estructuras de control (if, switch) y bucles (for, while).
    • Funciones de flecha (Arrow functions) y gestión del ámbito (Scope).

  2. Manipulación del DOM y Eventos
    • Selección, creación y modificación de elementos HTML.
    • Gestión de eventos del teclado, ratón y validación de formularios.
    • Modificación dinámica de clases y estilos CSS desde el código.

  3. Objetos, Arrays y ES6+
    • Métodos modernos de arrays (Map, filter, Reduce).
    • Introducción a la Programación Orientada a Objetos y Clases.
    • Desestructuración, Spread operator y módulos de JavaScript.

  4. Asincronía y Conectividad
    • Funcionamiento del Event Loop y uso de Callbacks.
    • Gestión de Promesas y sintaxis Async/Await.
    • Consumo de datos externos con Fetch API y procesamiento de JSON.
