Tarea 5.2 

Utilizando las etiquetas vistas en clase y los enlaces, intentar
crear un documento con ejemplos de cada tipo de etiqueta y enlazarlos
en un documento inicial parecido al enseñado en la clase.
Enlazar el documento creado en la tarea 5.1 también en la lista de enlaces.

Estructura del ejercicio:
- index.html: Documento índice que enlaza a los documentos de ejemplos de etiquetas.
- etiquetas_hx.html: Muestra como las etiquetas <hx> establecen diferentes jerarquías desde el nivel mas alto al mínimo.
- etiquetas_p.html: Muestra como las etiquetas <p> definen párrafos de texto.
- etiquetas_blockquote.html: Muestra como las etiquetas <blockquote> indican los bloques de texto que provienen de fuentes externas.
- etiquetas_cite.html: Muestra como las etiquetas <cite> se emplean para referenciar el título de un trabajo creativo (libros, películas, canciones, etc).
- etiquetas_em.html: Muestra como las etiquetas <em> indican énfasis en el texto (cursiva).
- em_vs_cite.html: Muestra que, aunque ambas se renderizan en cursiva, el motivo por el que usamos una u otra etiqueta es distinto.
- etiquetas_code.html: Muestra como las etiquetas <code> definen fragmentos de código de programación o marcado.
- etiquetas_strong.html: Muestra como las etiquetas <strong> identifican textos de gran importancia (negrita).
- etiquetas_de_hipervinculos.html: Muestra como las etiquetas <a> crean hipervínculos externos, internos o secciones del  mismo archivo.
- etiquetas_espacios_blanco.html: Muestra las diferentes etiquetas que controlan los espacios en blanco y saltos de línea.
- etiquetas_de_entidades.html: Muestra como se utilizan secuencias de caracteres para representar caracteres especiales o protegidos de HTML5.
