Tarea 5.2 

Utilizando las etiquetas vistas en clase y los enlaces, intentar
crear un documento con ejemplos de cada tipo de etiqueta y enlazarlos
en un documento inicial parecido al enseñado en la clase.
Enlazar el documento creado en la tarea 5.1 también en la lista de enlaces.

Estructura del ejercicio:
- index.html: Documento índice que enlaza a los documentos de ejemplos de etiquetas.
- etiquetas_hx.html: Muestra como las etiquetas <code><strong>&lt;hx&gt;</strong></code> establecen diferentes jerarquías desde el nivel mas alto al mínimo.
- etiquetas_p.html: Muestra como las etiquetas <code><strong>&lt;p&gt;</strong></code> definen párrafos de texto.
- etiquetas_blockquote.html: Muestra como las etiquetas <code><strong>&lt;blockquote&gt;</strong></code> indican los bloques de texto que provienen de fuentes externas.
- etiquetas_cite.html: Muestra como las etiquetas <code><strong>&lt;cite&gt;</strong></code> se emplean para referenciar el título de un trabajo creativo (libros, películas, canciones, etc).
- etiquetas_em.html: Muestra como las etiquetas <code><strong>&lt;em&gt;</strong></code> indican énfasis en el texto (cursiva).
- em_vs_cite.html: Muestra que, aunque ambas se renderizan en cursiva, el motivo por el que usamos una u otra etiqueta es distinto.
- etiquetas_code.html: Muestra como las etiquetas <code><strong>&lt;code&gt;</strong></code> definen fragmentos de código de programación o marcado.
- etiquetas_strong.html: Muestra como las etiquetas <code><strong>&lt;strong&gt;</strong></code> identifican textos de gran importancia (negrita).
- etiquetas_de_hipervinculos.html: Muestra como las etiquetas <code><strong>&lt;a&gt;</strong></code> crean hipervínculos externos, internos o secciones del  mismo archivo.
- etiquetas_espacios_blanco.html: Muestra las diferentes etiquetas que controlan los espacios en blanco y saltos de línea.
- etiquetas_de_entidades.html: Muestra como se utilizan secuencias de caracteres para representar caracteres especiales o protegidos de HTML5.
- docs: carpeta que contiene un documento .pdf para el enlace con descarga de etiquetas_de_hipervinculos.html
