La etiqueta <code> se usa para fragmentos de código fuente. Se representa con una fuente monoespaciada por defecto.

Si en Python defino una variable de esta manera saludo = "hello wrold" y quiero que el contenido de la variable se muestre en pantalla, hay que escribir print(saludo) en la terminal.
