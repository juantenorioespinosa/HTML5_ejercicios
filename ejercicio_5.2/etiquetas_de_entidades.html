Entidades en HTML5

Volver atrás

Algunos símbolos (como < o >) se usan para las etiquetas. Si los escribes tal cual, el navegador se confunde; para poder representarlos se usan las entidades.

Entidad &nbsp;

Crea un espacio manual que el navegador no ignora. Resultado: espacio en blanco

Entidad &lt;

Less than. Resultado: <

Entidad &gt;

Greater than. Resultado: >

Entidad &amp;

Ampersand. Resultado: &

Entidad &quot;

Double quotation mark. Resultado: "

Entidad &apos;

Single quotation mark (apostrophe). Resultado: '

Entidad &cent;

Cent. Resultado: ¢

Entidad &pound;

Pound. Resultado: £

Entidad &yen;

Yen. Resultado: ¥

Entidad &euro;

Euro. Resultado: €

Entidad &copy;

Copy. Resultado: ©

Entidad &reg;

Registered trademark. Resultado: ®
