Ejemplo de etiqueta <br>

La etiqueta <br> provoca un salto de línea manual justo en el lugar donde se inserta.

Lorem ipsum dolor
sit amet consectetur adipiscing elit quisque faucibus ex sapien vitae
pellentesque sem placerat in id cursus mi pretium tellus duis convallis tempus leo eu aenean sed diam urna tempor pulvinar vivamus fringilla lacus nec metus bibendum egestas iaculis massa nisl malesuada
lacinia integer nunc posuere ut hendrerit.

Ejemplo de etiqueta <wbr>

La etiqueta <wbr> indica una oportunidad de salto de línea. Es útil para que el navegador sepa por dónde romper una palabra muy larga si no cabe.

palaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaabramuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuuylaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaarga.

Ejemplo de etiqueta <pre>

    La etiqueta de bloque <pre> respeta todos los     espacios      y
    saltos
    de
    línea
    tal cual están escritos en el código.
    Se utiliza normalmente cuando queremos poner código fuente en nuestra web.

    Fragmento de código fuente:

    <-- Estructura básica de HTML -->
    
    <!DOCTYPE html>
    <html lang="en">

    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>HTML Tutorial</title>
    </head>

    <body>
        <h1>This is a heading</h1>
        <p>This is a paragraph.</p>
    </body>

    </html>
