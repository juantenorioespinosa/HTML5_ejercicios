Tarea 5.1 

Realizar un documento html usando la plantilla boilerplate del tema pasado.
Utiliza las etiquetas que hemos visto en esta clase para que tenga diferentes aspectos.

Etiquetas utilizadas en este ejercicio:
- <code><strong>&lt;h1&gt;</strong></code>: Para el título. Cada documento html debe incluir solo una etiqueta h1.
- <code><strong>&lt;h2&gt;</strong></code>: Para diferentes temas dentro del documento.
- <code><strong>&lt;strong&gt;</strong></code>: Marca texto como importante. El navegador lo pone en negrita y tiene un peso semántico alto.
- <code><strong>&lt;em&gt;</strong></code>: Para dar énfasis al texto. Se renderiza en cursiva y tiene menos relevancia semántica que strong.
- <code><strong>&lt;br&gt;</strong></code>: Provoca un salto de línea manual. Es una "etiqueta vacia" (no necesita cierre como </br>).
