Tarea 7.1: Etiquetas de Agrupación en HTML5

Este proyecto contiene una serie de ejercicios prácticos para dominar las etiquetas de agrupación y la creación de listas (ordenadas, desordenadas y de descripción).

Estructura del ejercicio:
- index.html: Menú principal con acceso a todos los ejercicios.
- lista_ordenada.html: Ejercicio 1 - Top 5 países por visitar.
- lista_invertida.html: Ejercicio 2 - Uso del atributo `reversed`.
- lista_desordenada.html: Ejercicio 3 - Películas favoritas.
- lista_numerada.html: Ejercicio 4.1 - Instrucciones de uso.
- lista_formateada.html: Ejercicio 4.2 - Menú de noticias anidado.
- lista_compleja_formateada.html: Ejercicio 4.3 - Réplica completa con listas de descripción (`<dl>`).

Enunciados:
1. Lista Ordenada: Crear una lista de 5 países usando números.
2. Lista Invertida: Invertir la numeración de la lista anterior.
3. Lista Desordenada: Crear una lista con viñetas de películas favoritas.
4. Réplica de Capturas: Maquetar tres documentos HTML para que coincidan exactamente con las imágenes de referencia, practicando la anidación de etiquetas.
